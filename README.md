# Todd Kozlowski - Professional Website

This is the source code for my personal professional website, built with Jekyll and hosted on GitHub Pages.

## Features

- Clean, professional design using the Jekyll Minimal Theme
- Responsive layout
- CV download link
- Professional headshot
- Easy to customize and maintain

## Local Development

To run this site locally:

```bash
bundle install
bundle exec jekyll serve
```

Then visit `http://localhost:4000` in your browser.

## Structure

- `_config.yml` - Jekyll configuration
- `index.md` - Home page content
- `assets/` - Images and documents (CV, headshot)

## Customization

Update the following in `_config.yml`:
- `title` - Your name
- `description` - Site description
- `author` - Your name
- `email` - Your email address
- `url` - Your site URL
